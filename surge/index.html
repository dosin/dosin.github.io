Surge Configuration

中

1 Download

If possible, buy surge in AppStore. Otherwise, Buy an APPID account that has already bought surge before or download it from some third party apps

2 Configure

2.1 Download Configuration from URL

https://dosin.github.io/surge.conf

2.2 Edit in Text Mode and find [Proxy]

Replace your own SS information in yellow parts

NAME = custom, HOST, PORT, METHOD, PASSWORD, https://dosin.github.io/SSEncrypt.module

Meanwhile, add the above NAME at the end of [Proxy Group] for convienence

Proxy = select, NAME

There you go

Surge配置

En

1 下载

如果可能，在AppStore购买surge。否则，购买一个以前买过surge的APPID或者从第三方软件处下载

2 配置

2.1 通过URL加载配置

Download Configuration from URL

https://dosin.github.io/surge.conf

2.2 Edit in Text Mode并且找到[Proxy]

用你的SS相关信息覆盖黄色部分

  • NAME：名称
  • HOST：服务器地址
  • PORT：服务器端口号
  • METHOD：加密方式
  • PASSWORD：密码
NAME = custom, HOST, PORT, METHOD, PASSWORD, https://dosin.github.io/SSEncrypt.module

同时为了便利，把上面的NAME加到[Proxy Group]后面

Proxy = select, NAME

好了

Dosin
